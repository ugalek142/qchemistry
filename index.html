Curso Profesional de Química Cuántica con Psi4

Módulo 1: Fundamentos de Química Cuántica

Teoría

La química cuántica estudia la estructura y propiedades de átomos y moléculas basándose en la mecánica cuántica. Es fundamental para comprender fenómenos a escala atómica y molecular.

La ecuación central es la ecuación de Schrödinger:

iħ ∂/∂t Ψ(r, t) = Ĥ Ψ(r, t)

donde:

  • Ψ: función de onda, que describe el estado cuántico del sistema.
  • Ĥ: operador Hamiltoniano, que representa la energía total del sistema.

Práctica

Para comenzar, instala Psi4, un potente software open-source para química computacional.

Ejecuta este código Python para calcular la energía del hidrógeno molecular (H₂) con un método básico:



import psi4

molecule = psi4.geometry("""
0 1
H
H 0.74
""")

energy = psi4.energy('scf/sto-3g', molecule=molecule)
print(f"Energía SCF: {energy} Hartree")

Tareas

Modifica la distancia entre átomos y observa cómo cambia la energía.

Investiga diferentes bases y métodos para mejorar la precisión.

Módulo 2: Introducción a Psi4

Teoría

Psi4 es un paquete avanzado para cálculos de química cuántica, que soporta métodos como HF, DFT, MP2 y CCSD.

Permite análisis detallados y automatización para simulaciones precisas.

Práctica

Ejemplo de cálculo de energía usando DFT con la base 6-31G*:



import psi4

molecule = psi4.geometry("""
0 1
H
H 0.74
""")

energy = psi4.energy('b3lyp/6-31g*', molecule=molecule)
print(f"Energía DFT: {energy} Hartree")

Tareas

Prueba diferentes funciones de densidad y analiza resultados.

Documenta las diferencias en energías y tiempos de cálculo.

Módulo 3: Cálculo de Energías

Teoría

El cálculo de energía es esencial para predecir estabilidad molecular y reactividad. Se utilizan métodos como Hartree-Fock (HF) y teoría del funcional de la densidad (DFT).

El resultado es la energía total del sistema, que incluye energía cinética y potencial.

Práctica

Calcula la energía de la molécula de agua (H₂O) usando MP2 con Psi4:



import psi4

molecule = psi4.geometry("""
0 1
O
H 1 0.96
H 1 0.96 2 104.5
""")

energy = psi4.energy('mp2/6-31g*', molecule=molecule)
print(f"Energía MP2: {energy} Hartree")

Tareas

Modifica el ángulo de enlace y observa su impacto en la energía.

Compara resultados usando bases diferentes y anota las diferencias.

Módulo 4: Optimización Geométrica

Teoría

La optimización geométrica busca la estructura molecular de mínima energía, ajustando posiciones atómicas.

Este proceso es crucial para identificar conformaciones estables y rutas de reacción.

Práctica

Ejecuta una optimización geométrica de la molécula de metano (CH₄):



import psi4

molecule = psi4.geometry("""
0 1
C
H 1 1.09
H 1 1.09 2 109.5
H 1 1.09 2 109.5 3 120
H 1 1.09 2 109.5 3 -120
""")

psi4.optimize('scf/6-31g*', molecule=molecule)

Tareas

Intenta optimizar otras moléculas simples como amoníaco o agua.

Compara tiempos y precisión usando diferentes métodos.

Módulo 5: Espectros y Propiedades

Teoría

Los cálculos espectroscópicos permiten predecir vibraciones, frecuencias y propiedades electrónicas.

Psi4 soporta cálculos de frecuencias vibracionales y análisis de propiedades moleculares.

Práctica

Ejemplo para calcular frecuencias vibracionales de H₂O:



import psi4

molecule = psi4.geometry("""
0 1
O
H 1 0.96
H 1 0.96 2 104.5
""")

psi4.set_options({'g_convergence': 'gau_tight'})
freqs = psi4.frequency('scf/6-31g*', molecule=molecule)

Tareas

Analiza los modos vibracionales y relaciona con espectros IR experimentales.

Realiza cálculos con otras moléculas pequeñas y compara resultados.

Módulo 6: Ejercicios Avanzados

Teoría

Este módulo combina todos los conocimientos previos para realizar cálculos complejos y análisis detallados.

Incluye optimización, cálculo de energías y propiedades en moléculas orgánicas e inorgánicas complejas.

Práctica

Ejemplo: Optimización y cálculo de energía para etileno (C₂H₄) usando DFT:



import psi4

molecule = psi4.geometry("""
0 1
C 0.0 0.0 0.0
C 1.34 0.0 0.0
H -0.67 0.93 0.0
H -0.67 -0.93 0.0
H 2.01 0.93 0.0
H 2.01 -0.93 0.0
""")

psi4.optimize('b3lyp/6-31g*', molecule=molecule)
energy = psi4.energy('b3lyp/6-31g*', molecule=molecule)
print(f"Energía DFT optimizada: {energy} Hartree")

Tareas

Realiza optimizaciones y cálculos de energía para diferentes moléculas orgánicas.

Compara resultados con métodos post-Hartree-Fock como MP2 y CCSD.

Prepara un informe detallado con gráficos y análisis.
